# HTML Site Generator Demo

This repository demonstrates two different approaches to generating static HTML sites:

1. Using Python (clean and minimal)
2. Using Java (verbose, class-heavy, conventional)

## Files

- `site.py` - Python implementation
- `HtmlSiteGenerator.java` - Java implementation

## Usage

### Python

```bash
python site.py
```

This will generate `my_site_python.html`.

### Java

```bash
javac HtmlSiteGenerator.java
java HtmlSiteGenerator
```

This will generate `my_site_java.html`.

## Comparison

The Python version follows a clean, functional approach with minimal boilerplate.
The Java version follows a more verbose, object-oriented approach with nested classes and builders.
